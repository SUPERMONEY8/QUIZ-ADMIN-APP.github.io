## Key Features

### 1. Member Management
- Add, edit, delete members
- 10 different groups
- Search and filter
- Member count tracking

### 2. Assignment Generation
- 6 different assignment algorithms
- Monthly generation with preview
- Transaction-based (all or nothing)
- Generation logging

### 3. Results & History
- Comprehensive results display
- Advanced filtering
- Export (CSV, PDF)
- Print functionality

### 4. Backup & Restore
- Manual and automatic backups

### 5. Settings
- Theme customization
- App preferences

## Technical Specifications

### Technology Stack
- **Runtime**: Electron 39.2.3
- **Package Manager**: Bun
- **Database**: SQLite (better-sqlite3 12.4.6)
- **Language**: JavaScript (ES6)
- **UI**: HTML5, CSS3

### System Requirements
- Windows 10/11 (64-bit)
- 1366x768 minimum resolution
- 100 MB free disk space
- No internet required

### Performance
- Generation: < 30s for 1000+ members
- Database: Optimized with indexes
- UI: Smooth scrolling, responsive
- Offline: Fully functional
