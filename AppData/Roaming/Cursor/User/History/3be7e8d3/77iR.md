# QUIZ-APP-ADMIN

Admin Access for Quiz App - This is the admin-access version that provides direct access to the admin dashboard **without requiring login**.

## Features

- Direct access to admin dashboard (no login required)
- Full admin functionality:
  - Create and manage quizzes
  - Add/edit questions
  - View analytics
  - Manual grading
  - All admin features

## Setup

1. Install dependencies:
```bash
bun install
# or
npm install
```

2. Set environment variables:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

3. Run development server:
```bash
bun run dev
# or
npm run dev
```

## Routes

- `/` - Admin dashboard
- `/admin` - Admin dashboard
- `/admin/new-quiz` - Create new quiz
- `/admin/quizzes/:quizId/questions` - Manage questions
- `/admin/quizzes/:quizId/questions/:questionId` - Edit question
- `/admin/results/:quizId` - View analytics
- `/admin/grading` - Manual grading

## Important Note

This version bypasses authentication. Use it only in secure environments or for development purposes.
