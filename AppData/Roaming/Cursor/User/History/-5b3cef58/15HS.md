# Audio Files Needed

## Required Audio Files

Please download and place the following audio files in the `public/audio/` folder:

### 1. Timer Ticking Sound
- **File**: `timer-tick.mp3` or `timer-tick.wav`
- **Description**: Subtle, soft ticking sound (like a clock tick)
- **Duration**: ~0.1-0.3 seconds
- **Volume**: Low/medium (will be played every 5 seconds)
- **Style**: Gentle, rhythmic, not too loud
- **Where to get**: 
  - Free: freesound.org (search "clock tick" or "timer tick")
  - Free: zapsplat.com (search "timer tick")
  - Free: mixkit.co (search "timer")

### 2. Timer Complete Sound
- **File**: `timer-complete.mp3` or `timer-complete.wav`
- **Description**: Soft, pleasant "ding" or bell sound when timer reaches 0
- **Duration**: ~0.5-1 second
- **Volume**: Medium
- **Style**: Gentle chime, bell, or soft notification sound (not harsh/alarming)
- **Where to get**:
  - Free: freesound.org (search "soft bell" or "gentle chime")
  - Free: zapsplat.com (search "notification bell")
  - Free: mixkit.co (search "notification")

### 3. Button Click Sound (Optional but recommended)
- **File**: `button-click.mp3` or `button-click.wav`
- **Description**: Soft click or tap sound for button interactions
- **Duration**: ~0.1-0.2 seconds
- **Volume**: Low
- **Style**: Subtle, soft tap/click
- **Where to get**:
  - Free: freesound.org (search "soft click" or "button tap")
  - Free: zapsplat.com (search "ui click")

### 4. Background Music (Optional)
- **File**: `background-music.mp3` or `background-music.wav`
- **Description**: Ambient, romantic, soft background music
- **Duration**: 2-5 minutes (will loop)
- **Volume**: Low (background level)
- **Style**: 
  - Soft, romantic instrumental
  - No vocals
  - Ambient, calming
  - Fits the red/brown dark love theme
- **Where to get**:
  - Free: pixabay.com/music (search "romantic ambient" or "soft instrumental")
  - Free: freemusicarchive.org
  - Free: bensound.com (royalty-free music)

## File Structure

Create this folder structure in your project:

```
Enjoyed/
├── public/
│   └── audio/
│       ├── timer-tick.mp3 (or .wav)
│       ├── timer-complete.mp3 (or .wav)
│       ├── button-click.mp3 (or .wav) [optional]
│       └── background-music.mp3 (or .wav) [optional]
```

## How to Provide Audio Files

### Option 1: Local Files (Recommended for Development)
1. Create the folder: `public/audio/`
2. Download the audio files from the sources above
3. Place them in `public/audio/` with the exact names:
   - `timer-tick.mp3`
   - `timer-complete.mp3`
   - `button-click.mp3` (optional)
   - `background-music.mp3` (optional)

### Option 2: Firebase Cloud Storage (For Production)
1. Upload files to Firebase Cloud Storage
2. Get the download URLs
3. Update the audio paths in the code (I'll show you where)

## Audio Format Recommendations

- **Format**: MP3 (smaller file size) or WAV (better quality)
- **Bitrate**: 128-192 kbps for MP3
- **Sample Rate**: 44.1 kHz
- **Channels**: Mono or Stereo (mono is smaller)

## Testing Without Audio Files

The app will work without audio files - it will just skip playing sounds if files are missing. You can add the audio files later.
