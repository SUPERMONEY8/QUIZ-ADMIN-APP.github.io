# Testing Guide - Supply Chain Management App

## Quick Start - Test Your App

### Method 1: Run the Compiled Executable (Fastest)
If you have a previously compiled version, you can run it directly:

1. **Double-click the executable:**
   - Navigate to: `bin\Debug\net8.0-windows\SupplyChainManagement.exe`
   - Double-click it to run

2. **Or use Command Prompt/PowerShell:**
   ```powershell
   cd "C:\Users\LENOVO\Desktop\Depo\bin\Debug\net8.0-windows"
   .\SupplyChainManagement.exe
   ```

### Method 2: Build and Run (Recommended)
If you need to rebuild the app:

1. **Use the batch file:**
   - Double-click `run.bat` in the project root
   - Or double-click `LAUNCH.bat`

2. **Use PowerShell:**
   ```powershell
   .\run.ps1
   ```

3. **Manual command line:**
   ```powershell
   dotnet restore
   dotnet build
   dotnet run
   ```

## Common Issues and Solutions

### Issue: App won't start
**Solution:**
- Check if .NET 8.0 Runtime is installed
- Verify database file permissions
- Check for build errors (see below)

### Issue: Build errors
**Solution:**
- The app currently has compilation errors
- Missing `using System.Threading.Tasks;` in interface files
- Missing `using System.Collections.Generic;` in interface files
- Need to fix these before building

### Issue: Database errors
**Solution:**
- Delete `SupplyChain.db` and let app recreate it
- Check file permissions
- Verify SQLite is working

### Issue: UI not displaying correctly
**Solution:**
- Check if WPF runtime is installed
- Verify XAML files are valid
- Check for missing resources

## Quick Test Script

Run this PowerShell script to quickly test if the app can start:

```powershell
# Test if executable exists and can run
$exePath = "bin\Debug\net8.0-windows\SupplyChainManagement.exe"
if (Test-Path $exePath) {
    Write-Host "Executable found! Starting app..." -ForegroundColor Green
    Start-Process $exePath
} else {
    Write-Host "Executable not found. Building app..." -ForegroundColor Yellow
    dotnet build
    if ($LASTEXITCODE -eq 0) {
        dotnet run
    } else {
        Write-Host "Build failed! Check errors above." -ForegroundColor Red
    }
}
```
