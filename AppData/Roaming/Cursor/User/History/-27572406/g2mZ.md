# Icons Directory

Place your application icons here:

- **app-icon.ico** - Main application icon (required for Windows)
- **icon.png** - Fallback icon (optional, used if .ico is not found)

The application will look for `app-icon.ico` first, and fall back to `icon.png` if the .ico file is not found.

## Icon Requirements

- **app-icon.ico**: Windows icon format, recommended sizes: 16x16, 32x32, 48x48, 256x256
- **icon.png**: PNG format, recommended size: 256x256 or higher

You can create icons using tools like:
- Online icon generators
- Image editing software (GIMP, Photoshop, etc.)
- Icon conversion tools
