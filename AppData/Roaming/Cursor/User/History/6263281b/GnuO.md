# Sound Effects

Place your audio files here for the signature animation:

## Required Files

1. **heartbeat.mp3** - Subtle heartbeat sound (looping)
   - Should be very quiet (volume ~0.1)
   - Duration: 1-2 seconds (will loop)
   - Format: MP3, WAV, or OGG

2. **chime.mp3** - Chime sound on hand touch
   - Pleasant, soft chime
   - Duration: 0.5-1 second
   - Format: MP3, WAV, or OGG

## Optional Files

- **spark.mp3** - Spark/explosion sound (if you want additional audio feedback)

## Free Audio Resources

You can find free audio files at:
- [Freesound.org](https://freesound.org)
- [Zapsplat](https://www.zapsplat.com)
- [Adobe Stock Audio](https://stock.adobe.com/audio)

## Note

If audio files are not provided, the animation will work without sound. The code handles missing audio files gracefully.
