# تطبيق المهام العسكرية - Military Assignment App

تطبيق سطح مكتب مبني باستخدام Electron و Bun لإدارة المهام العسكرية.

## المتطلبات

- Bun (الإصدار 1.0 أو أحدث)
- Windows 10/11

## التثبيت

```bash
# تثبيت التبعيات
bun install
```

## التشغيل

```bash
# تشغيل التطبيق
bun start
```

أو

```bash
bun run start
```

## المميزات

- ✅ يعمل 100% بدون اتصال بالإنترنت
- ✅ واجهة مستخدم باللغة العربية مع دعم RTL
- ✅ قاعدة بيانات SQLite محلية
- ✅ إدارة كاملة للمهام (إضافة، تعديل، حذف)
- ✅ تصفية المهام حسب الحالة
- ✅ نظام أولويات للمهام

## هيكل المشروع

```
military-assignment-app/
├── src/
│   ├── database/      # قاعدة البيانات SQLite
│   ├── styles/        # ملفات CSS
│   ├── scripts/       # ملفات JavaScript
│   └── pages/         # صفحات إضافية (اختياري)
├── assets/
│   ├── icons/         # أيقونات التطبيق
│   └── fonts/         # خطوط عربية (اختياري)
├── index.html         # الصفحة الرئيسية
├── main.js            # عملية Electron الرئيسية
├── preload.js         # سكريبت preload
├── package.json       # إعدادات المشروع
└── bunfig.toml        # إعدادات Bun
```

## التطوير

التطبيق يستخدم:
- **Electron**: لإطار عمل سطح المكتب
- **better-sqlite3**: لقاعدة البيانات المحلية
- **Bun**: كمدير حزم وبيئة تشغيل

## ملاحظات

- قاعدة البيانات تُنشأ تلقائياً في `src/database/app.db`
- جميع البيانات محلية ولا تحتاج اتصال بالإنترنت
- التطبيق جاهز للاستخدام على Windows
