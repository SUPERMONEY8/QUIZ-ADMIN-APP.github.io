## Getting Started

### First-Time Setup

1. **Launch the Application**
   - Double-click the Depo icon on your desktop
   - Or find it in your Start Menu under "Business" category

2. **Login**
   - Default username: `admin`
   - Default password: `admin123`
   - ⚠️ **Important**: Change your password immediately after first login!

3. **Configure Your Business**
   - Go to Settings
   - Enter your business information
   - Set your preferences

### Adding Your First Product

1. Navigate to **Products** from the sidebar
2. Click the **"Add Product"** button (top right)
3. Fill in the product details:
   - **Name**: Product name (required)
   - **Category**: Select or create a category
   - **Buying Price**: Cost price
   - **Selling Price**: Retail price
   - **Quantity**: Initial stock
   - **Unit**: Piece, Kg, Liter, etc.
   - **Barcode**: Optional barcode
4. Click **"Create Product"** to save

### Making Your First Sale

1. Go to **Sales** page
2. Click **"New Sale"** button
3. **Add Products to Cart**:
   - Search or browse products
   - Click on a product to add it
   - Adjust quantity if needed
4. **Select Customer** (optional):
   - Choose existing customer or create new one
5. **Choose Payment Method**:
   - **Full Payment**: Customer pays full amount
   - **Partial Payment**: Customer pays part now, rest later
   - **Credit**: Customer pays later (debt tracking)
6. Click **"Process Sale"** to complete

## Sales Management

### Creating a Sale

1. Click **"New Sale"** button
2. Add products to cart
3. Select customer (optional)
4. Choose payment method
5. Apply discount if needed
6. Complete the sale

### Payment Methods

#### Full Payment
- Customer pays the full amount immediately
- Sale is marked as "Paid"
- Invoice is generated

#### Partial Payment
- Customer pays a portion now
- Remaining amount tracked as debt
- Sale is marked as "Partial"
- Can collect remaining payment later

#### Credit (Deferred Payment)
- Customer pays later
- Full amount tracked as debt
- Sale is marked as "Pending"
- Set overdue date for tracking

### Sales History

- View all past sales
- Filter by date range
- Search by customer or product
- Print invoices
- View sale details

### Invoices

- Automatic invoice generation
- Print invoices
- Email invoices (future feature)
- Invoice numbering system

## Troubleshooting

### Application Won't Start

**Problem**: The application fails to launch or crashes on startup.

**Solutions**:
1. Check if you have the latest version installed
2. Try running as administrator
3. Check Windows Event Viewer for error details
4. Reinstall the application if the issue persists

### Data Not Saving

**Problem**: Changes are not being saved to the database.

**Solutions**:
1. Check file permissions for `%APPDATA%\Depo` folder
2. Ensure you have write permissions
3. Check disk space
4. Restart the application

### Slow Performance

**Problem**: The application is running slowly or freezing.

**Solutions**:
1. Close other applications to free up memory
2. Check if you have large amounts of data (consider archiving old records)
3. Restart the application
4. Check system resources in Task Manager

### Can't Find a Feature

**Problem**: You can't locate a specific feature.

**Solutions**:
1. Use the search bar to find features
2. Check the Help page (F1)
3. Review this manual
4. Contact support

## FAQ

### How do I backup my data?

Your data is automatically saved in `%APPDATA%\Depo\business.db.json`. You can manually copy this file to backup your data.

### Can I use Depo on multiple computers?

Yes, but you'll need to manually sync the database file between computers. Cloud sync is coming in a future update.

### How do I reset my password?

Go to Settings > Users, select your user, and click "Change Password".

### What payment methods are supported?

Depo supports Full Payment, Partial Payment, and Credit (deferred payment) options.

### How do I generate reports?

Go to Reports page, select the report type, choose date range, and click "Generate Report".

### Can I export data?

Yes, you can export data to CSV format from the Reports page. More export formats coming soon.

### How do I add a product image?

When creating or editing a product, click the image upload area and select an image file from your computer.

### Can I print invoices?

Yes, after completing a sale, you can click "Print Invoice" to print the invoice.

### How do I track customer debts?

When making a sale with Credit or Partial Payment, the debt is automatically tracked. View debts in the Customers page.

### What if I make a mistake in a sale?

You can edit or delete sales from the Sales page. However, be careful as this affects your inventory and financial records.
